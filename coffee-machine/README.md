# COFFEE MACHINE
A simply terminal version of a coffe machine.
## Installation
Clone the folder into yor computer and rum the following command on the terminal
```bash
python3 main.py
```
## Instructions

***```Choose your coffee```***

```bash
What would you like? (espresso/latte/cappuccino) 
```
